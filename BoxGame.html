Box Game

EASY -

Game 1

Winning color: BLUE

Non-Winning color: ORANGE

HOW TO PLAY: Assign to each of the symbols a color. An underlined symbol must have another color than the corresponding non-underlined symbol. If each box has at least one symbol colored with the winning color, you won!

PROOF: If you click at "Proof On" and a box is colored red, this box has no symbol colored with the winning color. If a symbol is colored red, have a look to the corresponding underlined or non-underlined symbol, does it have another color?
